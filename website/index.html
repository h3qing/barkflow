WhisperWoof

Speak and it transcribes, polishes, and routes — all locally.

Voice-first personal automation. Named after a good dog.

Download for macOS GitHub

What it does

🎙

Voice to polished text

Local Whisper STT + Ollama LLM removes filler words, fixes grammar, adds punctuation. Your voice never leaves your machine.

⌨️

Hotkey routing

Hold Fn to speak → text pastes at cursor. Cmd+K for command bar with /todo, /note, /project prefixes.

📋

Unified capture

Voice + clipboard history in one searchable view. Images too. Favorites, filters, full-text search.

🧩

MCP plugins

Route voice to Todoist, Notion, Slack, Calendar. Any MCP server works as a WhisperWoof plugin.

📁

Projects

Wandering mind capture. Funnel random thoughts into named buckets. Review and dispatch later.

🔒

Local-first

No cloud dependency. STT, LLM polish, and storage all run on your machine. Privacy by default.

How it works

1

Hold Fn

Press and hold the Globe/Fn key to start recording.

2

Speak

Say whatever you want. Mando's ears perk up while you talk.

3

Release

Text is transcribed, polished, and pasted at your cursor. Done.
