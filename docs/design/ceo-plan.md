> **Mando** is Heqing's dog — WhisperWoof's namesake and mascot. "Bark" = your voice into the mic. "Flow" = the workflow pipeline.

## Current State

Empty greenfield repo. No code, no commits. WhisperWoof will be a fork of [OpenWhispr](https://github.com/OpenWhispr/openwhispr) (Electron + React 19 + TypeScript + Tailwind v4, MIT licensed, 2K+ stars).

## Vision

### 10x Check
WhisperWoof as a Personal Automation OS: not just "voice triggers workflows" but an ambient capture layer that understands context, learns patterns, and becomes the single entry point for all personal automation. Raycast's command palette meets n8n's workflow engine, with voice as the primary input and MCP as the plugin protocol.

### Platonic Ideal
The user opens their laptop. WhisperWoof is in the menu bar. They hold Fn and say "Add milk to groceries." WhisperWoof transcribes, polishes, routes to the right list. They hold Fn+N and think out loud about a project — it saves as a polished Markdown note. They hold Fn+T and assign a task. Everything is in their searchable history. The floating dog-ear indicator perks up when they speak. It feels like a faithful assistant that just works.

## Scope Decisions

| # | Proposal | Effort | Decision | Reasoning |
|---|----------|--------|----------|-----------|
| 1 | LLM-based intent routing (one hotkey, LLM detects destination) | M | DEFERRED | User wants explicit hotkey-driven intent in Phase 1 — more predictable, more customizable |
| 2 | Command palette UI (Raycast-style launcher) | L | DEFERRED | Good vision but not Phase 1. Start with menu bar + hotkeys. |
| 3 | MCP as the plugin system (plugins are MCP servers) | M | ACCEPTED (Phase 2) | User explicitly mentioned "different MCPs" in vision. MCP is a published standard — no reason to build custom. |
| 4 | Voice-to-Markdown notes (Fn+N → speak → .md file) | S | ACCEPTED (Phase 1) | Trivial cost, high delight. Natural extension of hotkey-driven routing. |
| 5 | Audio playback in history + cache management | S | ACCEPTED (Phase 1) | Transforms history from text log to voice journal. User requested cache management for storage control. |
| 6 | Branded floating indicator (Classic + Bark dog ear styles) | S | ACCEPTED (Phase 1) | Brand touch that gets shared. Mando deserves it. |

## Explicitly Out of Scope
- Multi-platform (Windows/Linux) — Electron supports it but not Phase 1 priority
- Cloud sync / collaborative features
- Mobile app
- Multi-language STT (English only for MVP)
- LLM intent detection (hotkey-driven routing only)
- Commercial/paid features

## Success Criteria
See design doc for full list. Key targets:
- Use WhisperWoof daily instead of OpenWhispr
- LLM polish <1 second on Apple Silicon M1+
- Works correctly when Ollama is not installed (graceful degradation)
- At least 2 working MCP plugins route voice to action
- Clipboard + voice history searchable and useful
- A productivity nerd friend says "whoa" when you demo it

## Risks & Mitigations

| Risk | Likelihood | Impact | Mitigation |
|------|-----------|--------|------------|
| OpenWhispr upstream breaks fork compatibility | Med | Med | Keep WhisperWoof additions in separate modules, minimize edits to core files |
| Ollama latency too slow on smaller Macs | Low | High | Benchmark models on M1, offer "skip polish" option, use smallest viable model (3B) |
| MCP protocol changes (still evolving) | Low | Med | Pin MCP SDK version, abstract behind interface |
| macOS Fn key intercepted by system | Low | Med | Allow fully configurable hotkeys, not just Fn combos |
| Electron memory overhead | Med | Low | Acceptable trade-off for cross-platform compatibility. Tauri rewrite is a backlog option. |

## Key Design Principles (from user)
1. **Hotkey = intent.** The key combination determines the destination, not the LLM. Explicit > smart.
2. **Local-first is non-negotiable.** Everything runs on user's machine. Cloud is a future paid tier.
3. **Repository pattern for storage.** Abstract from day one for future cloud/bring-your-own-backend providers.
4. **Fork, don't reinvent.** Build on OpenWhispr's proven STT and Electron shell.
